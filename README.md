# Recommendation_system
Recommender system is an essential tool that can suggest personalized products or services for a given customer and directly help to increase business sales and attract more customers. Recommender systems help users select similar items when something is being chosen online. Companies such as Netflix or Amazon, would suggest users different movies that might interest them and are worth watching. There are 2 techniques in Recommendation system. **Content-based filtering**, makes recommendations based on user preferences for product features. **Collaborative filtering** mimics user-to-user recommendations
